# Microposts

Simple application using babel

### Version
1.0

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm start
```

You also need to install an run json-server(https://github.com/typicode/json-server)

### Build
Compile and build

```sh
$ npm run build
```
